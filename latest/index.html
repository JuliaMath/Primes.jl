Primes.jl

This package provides functions for computing prime numbers in Julia.

Installation

The package is available for Julia versions 0.4 and up, but if you are not using Julia version 0.7 or higher, please read the documentation for the appropriate realease.

To install it, run

Pkg.add("Primes")

from the Julia REPL.
